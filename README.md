# message-broker-benchmark
Benchmarking RabbitMQ, Redis, and Kafka in Golang for throughput, latency, and fault tolerance. Inspired by real-world peak-demand systems.

## Setup
- Requirements: Golang 1.21+, Docker.
- Run: `docker-compose up -d`

## Scenarios
- Simple Queueing
- High-Concurrency Load
- Failure Recovery
- Pub/Sub Broadcast
